### Installation
1. Install dependences.
    ```bash
    npm install
    ```
2. Copy the config.json, and edit content.
    ```bash
    cp config.sample.json config.json
    ```
3. Build.
    ```bash
    npm run build
    ```
4. Then you can found the result in `build/index.user.js`.
